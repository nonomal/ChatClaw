5 मिनट में OpenClaw जैसा व्यक्तिगत AI एजेंट प्राप्त करें। सैंडबॉक्स सुरक्षित, macOS और Windows के लिए अति-छोटा 30MB इंस्टॉलर (1 मिनट में इंस्टॉल)। WhatsApp, Telegram, Slack, Discord, Gmail, DingTalk, WeChat Work, QQ, Feishu और अन्य मैसेजिंग ऐप्स से कनेक्ट होता है। बिल्ट-इन स्किल मार्केट, नॉलेज बेस, मेमोरी, MCP, शेड्यूल्ड टास्क। Go में विकसित: तेज़ और कम संसाधन उपयोग।

## पूर्वावलोकन

### AI Chatbot Assistant

अपने AI सहायक से कोई भी सवाल पूछें; यह बुद्धिमानी से आपके ज्ञान आधार को खोजेगा और प्रासंगिक उत्तर उत्पन्न करेगा।内置 স্কিল মার্কেটের সাথে, AI এজেন্টগুলি স্বায়ত্তশাসিতভাবে কাজ করते हैं — पूरे समय ম্যানুয়াল তদারকির প্রয়োজন নেই। জটিল প্রজেক্ট প্ল্যানিং, দস্তাবেজ সংগঠন, PPT জেনারেশন বা মাল্টি-স্টেপ টাস্ক এক্সিকিউশনে, AI এজেন্টগুলি স্বায়ত্তশাসিতভাবে বিশ্লেষণ করতে, দক্ষতার সাথে এগিয়ে যেতে এবং চূড়ান্তভাবে সম্পূর্ণ ফলাফল দিতে পারে।

### Multi-Agent Mode, Tailored for Different Scenarios

একাধিক স্বাধীন AI এজেন্ট তৈরি করুন, প্রতিটির নিজস্ব ব্যক্তিত্ব, স্মৃতি এবং দক্ষতা রয়েছে। প্রয়োজন অনুযায়ী হস্তক্ষেপ ছাড়াই তাদের মধ্যে স্যুইচ করুন। প্রতিটি এজেন্টকে বিভিন্ন ভূমিকা (যেমন "গ্রাহক সেবা", "ডেটা বিশ্লেষক", "কপি অ্যাসিস্ট্যান্ট") দেওয়া যায় এবং বিভিন্ন দক্ষতা, জ্ঞান আधार এবং প্রতিক্রিয়া শৈলী দিয়ে কনফিগার করা যায়।

### Local Knowledge Base Management

আপনার নথি আপলোড করুন (TXT, PDF, Word, Excel, CSV, HTML, Markdown)। সিস্টেম স্বয়ংক্রিয়ভাবে পার্স, বিভক্ত এবং ভেক্টর এম্বেডিংয়ে রূপান্তর করে আপনার ব্যক্তিগত জ্ঞান আধারে সংরক্ষণ করে AI দ্বারা সুনির্দিষ্ট পুনরুদ্ধার এবং ব্যবহারের জন্য। ফোল্ডার এবং জ্ঞান আধার অনুযায়ী নথি সংগঠন সমর্থন করে।

## सर्वर मोड परिनियोजन

ChatClaw सर्वर मोड में चला सकता है (डेस्कटॉप GUI की आवश्यकता नहीं), ब्राउज़र के माध्यम से सुलभ।

### बाइनरी सीधे चलाएं

अपने प्लेटफॉर्म के लिए बाइनरी [GitHub Releases](https://github.com/chatwiki/chatclaw/releases) से डाउनलोड करें:

|| प्लेटफॉर्म | फ़ाइल |
||----------|------|
|| Linux x86_64 | `ChatClaw-server-linux-amd64` |
|| Linux ARM64 | `ChatClaw-server-linux-arm64` |

```bash
chmod +x ChatClaw-server-linux-amd64
./ChatClaw-server-linux-amd64
```

अपने ब्राउज़र में http://localhost:8080 खोलें।

सर्वर डिफ़ॉल्ट पर `0.0.0.0:8080` पर सुनता है। आप पर्यावरण चर के माध्यम से होस्ट और पोर्ट कस्टमाइज़ कर सकते हैं:

```bash
WAILS_SERVER_HOST=127.0.0.1 WAILS_SERVER_PORT=3000 ./ChatClaw-server-linux-amd64
```

### Docker

```bash
docker run -d \
  --name chatclaw-server \
  -p 8080:8080 \
  -v chatclaw-data:/root/.config/chatclaw \
  registry.cn-hangzhou.aliyuncs.com/chatwiki/chatclaw:latest
```

अपने ब्राउज़र में http://localhost:8080 खोलें।

### Docker Compose

`docker-compose.yml` फ़ाइल बनाएं:

```yaml
services:
  chatclaw:
    image: registry.cn-hangzhou.aliyuncs.com/chatwiki/chatclaw:latest
    container_name: chatclaw-server
    volumes:
      - chatclaw-data:/root/.config/chatclaw
    ports:
      - "8080:8080"
    restart: unless-stopped

volumes:
  chatclaw-data:
```

फिर चलाएं:

```bash
docker compose up -d
```

अपने ब्राउज़र में http://localhost:8080 खोलें। रोकने के लिए: `docker compose down`। डेटा `chatclaw-data` वॉल्यूम में रहता है।
