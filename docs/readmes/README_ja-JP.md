ChatClawはオープンソースのローカルナレッジベースとOpenClawグラフicalデスクトップアシスタントアプリケーションです。
プログラミング不要でワンクリックによりローカルPCにデプロイ可能。微信、 釘钉、企業微信、QQ、飛書、WhatsAppなどの主流通信アプリに接続でき、
コマンドを送信するだけでAIにタスクを実行してもらえます。内置 OpenClaw 5000+ スキルライブラリを持ち、imaに似たローカルナレッジベース管理もサポートしています。

## プレビュー

### AI Chatbot Assistant

AIアシスタントに質問すれば、知識ベースをインテリジェントに検索し、関連する回答を生成します。組み込みスキルマーケットにより、AIエージェントは自律的に動作し，手動の監視は必要ありません。複雑なプロジェクト計画、ドキュメント整理、PPT生成、 멀티ステップタスク実行のいずれにおいても、AIエージェントは自律的に分析し、効率的に推進し、最終的に完全な結果を達成します。

![](../../images/previews/en/image1.png)

![](../../images/previews/en/image2.png)

### Multi-Agent Mode, Tailored for Different Scenarios

それぞれ独自のペルソナ、メモリースキルを持つ複数の独立したAIエージェントを作成します。需要に応じて互いに干涉せずに切り替えます。各エージェントには「カスタマーサービス」「データアナリスト」「コピーアシスタント」などの異なる役割を割り当て、異なるスキル、知識ベース、応答スタイルで構成できます。

![](../../images/previews/en/image3.png)

### Local Knowledge Base Management

ドキュメント（TXT、PDF、Word、Excel、CSV、HTML、Markdown）をアップロードします。システムは自動的に解析、分割、ベクトル化し、プライベートナレッジベースに保存して、AIによる精密な検索と利用を可能にします。ドキュメントをフォルダとナレッジベースで整理することをサポートします。

![](../../images/previews/en/image4.png)

### Skill Manager — AI Responds at Command

5000以上のすぐに使えるAIスキルが生産性、開発ツール、マルチメディア、スマートホームなどをカバーします。プログラミングなしでAIに作業を任せます。コマンドを使用して、PCにインストール済みの機能を見つけたり、新しい拡張プラグインをインストールしたりできます。スキルマーケットでスキルを自由に閲覧してインストールできます。

![](../../images/previews/en/image5.png)

### Memory — Natural & Smarter Interactions

コンテキスト対話を有効化し、パーソナライズされたサービスを提供し、複雑なタスクを完了させ、継続的な学習と進化を実現します。ロボットは、より思慮深く、より知的なサービスを提供する成長するパートナーになります。

## サーバーモードデプロイ

ChatClawはサーバーモードで実行可能（デスクトップGUI不要）、ブラウザでアクセス可能。

### バイナリ直接実行

[GitHub Releases](https://github.com/chatwiki/chatclaw/releases)から該当プラットフォームのバイナリをダウンロード：

|| プラットフォーム | ファイル |
||------|------|
|| Linux x86_64 | `ChatClaw-server-linux-amd64` |
|| Linux ARM64 | `ChatClaw-server-linux-arm64` |

```bash
chmod +x ChatClaw-server-linux-amd64
./ChatClaw-server-linux-amd64
```

ブラウザで http://localhost:8080 を開く。

デフォルトで `0.0.0.0:8080` をリッスン。環境変数でホストとポートをカスタマイズ可能：

```bash
WAILS_SERVER_HOST=127.0.0.1 WAILS_SERVER_PORT=3000 ./ChatClaw-server-linux-amd64
```

### Docker

```bash
docker run -d \
  --name chatclaw-server \
  -p 8080:8080 \
  -v chatclaw-data:/root/.config/chatclaw \
  registry.cn-hangzhou.aliyuncs.com/chatwiki/chatclaw:latest
```

ブラウザで http://localhost:8080 を開く。

### Docker Compose

`docker-compose.yml` ファイルを作成：

```yaml
services:
  chatclaw:
    image: registry.cn-hangzhou.aliyuncs.com/chatwiki/chatclaw:latest
    container_name: chatclaw-server
    volumes:
      - chatclaw-data:/root/.config/chatclaw
    ports:
      - "8080:8080"
    restart: unless-stopped

volumes:
  chatclaw-data:
```

その後実行：

```bash
docker compose up -d
```

ブラウザで http://localhost:8080 を開く。停止：`docker compose down`。データは `chatclaw-data` ボリュームに永続化。
