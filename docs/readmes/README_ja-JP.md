OpenClawのような個人AIエージェントを5分で入手。サンドボックスセキュリティ付き、macOS＆Windows用の超小型30MBインストーラー（1分でインストール）。WhatsApp、Telegram、Slack、Discord、Gmail、DingTalk、WeChat Work、QQ、Feishuなどのメッセージングアプリに接続。組み込みスキルマーケット、ナレッジベース、メモリ、MCP、スケジュールタスク。Goで開発：高速でリソース使用量が少ない。

## プレビュー

### AI チャットアシスタント
AI アシスタントに何でも質問してください。アシスタントはお客様の知識ベースをインテリジェントに検索し、関連する回答を生成します。
![](../../images/previews/en/image1.png)

### デュアルモード切り替え、効率的なタスク処理
チャットモードはマルチシナリオの Q&A と推論に適応；タスクモードは内置スキルマーケットと連携し、AI エージェントが自律的にマルチステップタスクを分解・推進し、効率を向上させます。
![](../../images/previews/en/image2.png)

### PPT 即座に生成
スマートアシスタントに一文のコマンドを送信すると、PowerPoint プレゼンテーションを自動作成・生成します。
![](../../images/previews/en/image3.png)

### スキルマネージャー
コマンドでアシスタントに、计算机にインストールされた機能の検索や新しい拡張プラグインのインストールを依頼できます。スキルマーケット — 自由にスキルを閲覧・インストール。
![](../../images/previews/en/image4.png)

### メモリ — より自然な、よりスマートな対話
コンテキスト会話を実現し、パーソナライズされたサポートを提供します。継続的な学習と進化 — アシスタントはますます思慮深くインテリジェントなサービスを提供する成長するパートナーのように感じます。
![](../../images/previews/en/image5.png)

### 無料モデル試用 — 共有チームナレッジベース
ワンクリックで ChatWiki に接続認可し、ChatWiki アカウントクレジットを同期、カスタムモデルもサポート。内蔵の高品質海内外 LLMs（Ollama、Google Gemini、OpenAI など）を搭載 — 日常業務でもプロフェッショナルなシナリオでも、お好みの AI モデルを使用できます。
![](../../images/previews/en/image6.png)

### ナレッジベース | ドキュメントベクトル化ストレージ
ドキュメント（TXT、PDF、Word、Excel、CSV、HTML、Markdown）をアップロードすると、システムは自動的に解析・分割・ベクトル化し、プライベートナレッジベースに保存。AI モデルによる精密な検索・活用を実現します。
![](../../images/previews/en/image7.png)

### 豊かな IM チャンネル連携
即時通信サービスプロバイダー（Feishu、WeCom、QQ、DingTalk、WeChat、WhatsApp など）が提供する SDK を統合することで、チャネル作成、ユーザー管理、メッセージ送受信を含む完全な IM 通信機能をアプリ内で素早く実装できます。
![](../../images/previews/en/image8.png)

### スケジュールタスク — 自動化コマンド実行
アシスタントに事前に設定した時刻や間隔で特定の操作を自動的に実行させます。例えば、タイムリーなリマインダーの提供、周期的な仕事の実行、システムレベルのメンテナンスタスクの実行など。
![](../../images/previews/en/image9.png)

### テキスト選択で即座に Q&A
画面上の任意のテキストを選択すると、自動的にフローティングのクイック質問ボックスにコピーされます。ワンクリックで AI アシスタントに送信し、即座に回答を獲得。
![](../../images/previews/en/image10.png)

### スマートサイドバー
他のアプリウィンドウにスナップできるスマートアシスタント。異なる設定の AI アシスタント間を素早く切り替えて質問できます。ロボットは関連するナレッジベースに基いて回答を生成し、クリック一回で返信を会話に送信することもサポート。
![](../../images/previews/en/image11.png)

### 一問多答：簡単に比較
質問を繰り返す必要がありません。複数の「AI 専門家」に同時に相談し、同じインターフェースで回答を並列表示。比較が簡単で、最良の結論に達するのをサポートします。
![](../../images/previews/en/image12.png)

### ワンクリックランチャーボール
デスクトップのフローティングボールをクリックすれば、ChatClaw メインアプリウィンドウを即座に起動または開きます。
![](../../images/previews/en/image13.png)

## サーバーモードデプロイ

ChatClawはサーバーモードで実行可能（デスクトップGUI不要）、ブラウザでアクセス可能。

### バイナリ直接実行

[GitHub Releases](https://github.com/chatwiki/chatclaw/releases)から該当プラットフォームのバイナリをダウンロード：

|| プラットフォーム | ファイル |
||------|------|
|| Linux x86_64 | `ChatClaw-server-linux-amd64` |
|| Linux ARM64 | `ChatClaw-server-linux-arm64` |

```bash
chmod +x ChatClaw-server-linux-amd64
./ChatClaw-server-linux-amd64
```

ブラウザで http://localhost:8080 を開く。

デフォルトで `0.0.0.0:8080` をリッスン。環境変数でホストとポートをカスタマイズ可能：

```bash
WAILS_SERVER_HOST=127.0.0.1 WAILS_SERVER_PORT=3000 ./ChatClaw-server-linux-amd64
```

### Docker

```bash
docker run -d \
  --name chatclaw-server \
  -p 8080:8080 \
  -v chatclaw-data:/root/.config/chatclaw \
  registry.cn-hangzhou.aliyuncs.com/chatwiki/chatclaw:latest
```

ブラウザで http://localhost:8080 を開く。

### Docker Compose

`docker-compose.yml` ファイルを作成：

```yaml
services:
  chatclaw:
    image: registry.cn-hangzhou.aliyuncs.com/chatwiki/chatclaw:latest
    container_name: chatclaw-server
    volumes:
      - chatclaw-data:/root/.config/chatclaw
    ports:
      - "8080:8080"
    restart: unless-stopped

volumes:
  chatclaw-data:
```

その後実行：

```bash
docker compose up -d
```

ブラウザで http://localhost:8080 を開く。停止：`docker compose down`。データは `chatclaw-data` ボリュームに永続化。
