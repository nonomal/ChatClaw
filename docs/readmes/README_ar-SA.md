احصل على وكيل AI شخصي يشبه OpenClaw في 5 دقائق. مؤمّن بـ Sandbox، مع برنامج تثبيت صغير للغاية 30 ميجابايت لـ macOS و Windows (تثبيت في دقيقة واحدة). يتصل بـ WhatsApp وTelegram وSlack وDiscord وGmail وDingTalk وWeChat Work وQQ وFeishu وتطبيقات المراسلة الأخرى. يحتوي على سوق المهارات وقاعدة المعرفة والذاكرة وMCP والمهام المجدولة. مطوّر بـ Go: سريع واستهلاك منخفض للموارد.

## المعاينة

### AI Chatbot Assistant

اطرح أي سؤال على مساعدك الذكي، وسوف يبحث في قاعدة معارفك بذكاء ويقدم إجابات ذات صلة. مع سوق المهارات المدمج، يعمل وكلاء الذكاء الاصطناعي بشكل مستقل — لا حاجة للإشراف اليدوي طوال الوقت. سواء كان تخطيطًا معقدًا للمشروع أو تنظيم المستندات أو إنشاء عروض PowerPoint أو تنفيذ المهام المتعددة الخطوات، يمكن للوكلاء المستقلين التحليل الذكي والتقدم بكفاءة، وفي النهاية تقديم نتائج كاملة، مما يوفر الكثير من الوقت.

![](../../images/previews/en/image1.png)

![](../../images/previews/en/image2.png)

### Multi-Agent Mode, Tailored for Different Scenarios

أنشئ وكلاء ذكاء اصطناعي مستقلين متعددين، لكل منهم شخصيته وذاكرة ومهاراته الخاصة. قم بالتبديل بينهم حسب الحاجة دون تدخل متبادل. يمكن تعيين أدوار مختلفة لكل وكيل (مثل "خدمة العملاء" و"محلل البيانات" و"مساعد النسخ") وتكوين مهارات وقواعد معارف وأساليب استجابة مختلفة.

![](../../images/previews/en/image3.png)

### Local Knowledge Base Management

قم بتحميل مستنداتك (TXT وPDF وWord وExcel وCSV وHTML وMarkdown). يحلل النظام تلقائيًا ويفصل ويحولها إلى تضمينات متجهية مخزنة في قاعدة معارفك الخاصة لاسترجاع واستخدام دقيق بواسطة الذكاء الاصطناعي. يدعم تنظيم المستندات حسب المجلد وقاعدة المعرفة.

![](../../images/previews/en/image4.png)

### Skill Manager — AI Responds at Command

5000+ مهارة ذكاء اصطناعي جاهزة للاستخدام تغطي الإنتاجية وأدوات التطوير والوسائط المتعددة والمنزل الذكي والمزيد — مما يسمح للذكاء الاصطناعي بالعمل من أجلك دون برمجة. استخدم الأوامر لمساعدة الروبوت في البحث عن الميزات المثبتة أو تثبيت ملحقات جديدة. سوق المهارات — تصفح وثبّت المهارات بحرية.

![](../../images/previews/en/image5.png)

### Memory — Natural & Smarter Interactions

تمكين المحادثات السياقية، وتقديم خدمات مخصصة، وإكمال المهام المعقدة، والتعلم المستمر والتطور. يصبح الروبوت رفيقًا متناميًا يقدم خدمة أكثرThoughtful وذكاءً.

![](../../images/previews/en/image6.png)

### Free Model Trial

ترخيص بنقرة واحدة للاتصال بـ ChatWiki ومزامنة الأرصدة ودعم النماذج المخصصة. نماذج لغوية كبيرة عالية الجودة مدمجة: Ollama وGoogle Gemini وOpenAI والمزيد — استخدم نموذج الذكاء الاصطناعي المفضل لديك للعمل المكتبي أو السيناريوهات المهنية.

![](../../images/previews/en/image7.png)

### Multi-Channel Remote Control via WeChat/QQ/WhatsApp

يدعم ChatClaw قنوات رسائل متعددة، ويدفع نتائج التحليل وتنبيهات المراقبة وملخصات البحث مباشرة إلى هاتفك. بعد معالجة الذكاء الاصطناعي، يتم إرسال النتائج تلقائيًا إلى القناة المحددة. أرسل الأوامر في نافذة الدردشة للتحكم عن بعد في تنفيذ مهام الذكاء الاصطناعي.

![](../../images/previews/en/image8.png)

### Scheduled Tasks — Automated Execution

حدد تردد المراقبة: كل 5 دقائق أو كل ساعة أو يوميًا في أوقات محددة. جدولة رسومية مع تعبيرات cron. جلب الصفحات أو مصادر البيانات بشكل دوري، ومقارنة التغييرات، ومراقبة المؤشرات الرئيسية، ودفع التنبيهات عبر قنوات الرسائل عند اكتشاف الشذوذ.

![](../../images/previews/en/image9.png)

### Text Selection for Instant Q&A

حدد أي نص على الشاشة؛ يتم نسخه تلقائيًا في مربع السؤال السريع العائم. انقر مرة واحدة لإرساله إلى مساعد الذكاء الاصطناعي للحصول على إجابة فورية.

![](../../images/previews/en/image10.png)

### Smart Sidebar

مساعد ذكي يلتصق بجانب نوافذ التطبيقات الأخرى. قم بالتبديل بسرعة بين مساعدي الذكاء الاصطناعي المتكوينين بشكل مختلف لطرح الأسئلة. يولد الروبوت إجابات بناءً على قاعدة معارفك المرتبطة ويدعم إرسال الرد بنقرة واحدة. المتابعة العائمة الذكية — إدخال الأداة دائمًا في متناول اليد.

انقر على الكرة العائمة على سطح المكتب الخاص بك لاستيقاظ أو فتح نافذة تطبيق ChatClaw الرئيسي فورًا.

## نشر وضع الخادم

يمكن لـ ChatClaw العمل كخادم (لا حاجة لواجهة رسومية للسطح المكتب)، ويمكن الوصول إليه عبر المتصفح.

### الثنائي المباشر

قم بتنزيل الثنائي لمنصتك من [GitHub Releases](https://github.com/chatwiki/chatclaw/releases):

|| المنصة | الملف |
||----------|------|
|| Linux x86_64 | `ChatClaw-server-linux-amd64` |
|| Linux ARM64 | `ChatClaw-server-linux-arm64` |

```bash
chmod +x ChatClaw-server-linux-amd64
./ChatClaw-server-linux-amd4
```

افتح http://localhost:8080 في متصفحك.

الخادم يستمع على `0.0.0.0:8080` افتراضيًا. يمكنك تخصيص المضيف والمنفذ عبر متغيرات البيئة:

```bash
WAILS_SERVER_HOST=127.0.0.1 WAILS_SERVER_PORT=3000 ./ChatClaw-server-linux-amd64
```

### Docker

```bash
docker run -d \
  --name chatclaw-server \
  -p 8080:8080 \
  -v chatclaw-data:/root/.config/chatclaw \
  registry.cn-hangzhou.aliyuncs.com/chatwiki/chatclaw:latest
```

افتح http://localhost:8080 في متصفحك.

### Docker Compose

أنشئ ملف `docker-compose.yml`:

```yaml
services:
  chatclaw:
    image: registry.cn-hangzhou.aliyuncs.com/chatwiki/chatclaw:latest
    container_name: chatclaw-server
    volumes:
      - chatclaw-data:/root/.config/chatclaw
    ports:
      - "8080:8080"
    restart: unless-stopped

volumes:
  chatclaw-data:
```

ثم شغّل:

```bash
docker compose up -d
```

افتح http://localhost:8080 في متصفحك. للإيقاف: `docker compose down`. البيانات موجودة بشكل مستمر في حجم `chatclaw-data`.
