ChatClaw হল একটি ওপেন সোর্স লোকাল নলেজ বেস এবং OpenClaw গ্রাফিক্যাল ডেস্কটপ অ্যাসিস্ট্যান্ট অ্যাপ্লিকেশন।
কোনো প্রোগ্রামিং প্রয়োজন নেই — এক ক্লিকে আপনার স্থানীয় কম্পিউটারে ডেপ্লয় করুন। WeChat, DingTalk, WeCom, QQ, Feishu, WhatsApp এবং অন্যান্য প্রধান মেসেজিং অ্যাপের সাথে সংযুক্ত হন।
কমান্ড পাঠান এবং AI কে আপনার জন্য কাজ করতে দিন। OpenClaw 5000+ দক্ষতা লাইব্রেরি সহ অন্তর্নির্মিত, IMA-র মতো স্থানীয় নলেজ বেস ব্যবস্থাপনা সমর্থিত।

## প্রিভিউ

### AI Chatbot Assistant

আপনার AI অ্যাসিস্ট্যান্টকে যেকোনো প্রশ্ন জিজ্ঞাসা করুন; এটি বুদ্ধিমানভাবে আপনার জ্ঞান ভিত্তি অনুসন্ধান করবে এবং প্রাসঙ্গিক উত্তর তৈরি করবে।内置 স্কিল মার্কেটের সাথে, AI এজেন্টগুলি স্বায়ত্তশাসিতভাবে কাজ করে — সম্পূর্ণ প্রক্রিয়ায় ম্যানুয়াল তদারকির প্রয়োজন নেই। জটিল প্রজেক্ট পরিকল্পনা, নথি সংগঠন, PPT তৈরি বা বহু-পদক্ষেপ কাজের ক্ষেত্রে, AI এজেন্টগুলি স্বায়ত্তশাসিতভাবে বিশ্লেষণ করতে, দক্ষতার সাথে এগিয়ে যেতে এবং চূড়ান্তভাবে সম্পূর্ণ ফলাফল প্রদান করতে পারে।

### Multi-Agent Mode, Tailored for Different Scenarios

একাধিক স্বাধীন AI এজেন্ট তৈরি করুন, প্রতিটির নিজস্ব ব্যক্তিত্ব, স্মৃতি এবং দক্ষতা রয়েছে। প্রয়োজনে হস্তক্ষেপ ছাড়াই তাদের মধ্যে স্যুইচ করুন। প্রতিটি এজেন্টকে বিভিন্ন ভূমিকা (যেমন "গ্রাহক সেবা", "ডেটা বিশ্লেষক", "কপি অ্যাসিস্ট্যান্ট") দেওয়া যায় এবং বিভিন্ন দক্ষতা, জ্ঞান ভিত্তি এবং প্রতিক্রিয়া শৈলী দিয়ে কনফিগার করা যায়।

### Local Knowledge Base Management

আপনার নথি আপলোড করুন (TXT, PDF, Word, Excel, CSV, HTML, Markdown)। সিস্টেম স্বয়ংক্রিয়ভাবে পার্স, বিভক্ত এবং ভেক্টর এম্বেডিংয়ে রূপান্তর করে আপনার ব্যক্তিগত নলেজ বেসে সংরক্ষণ করে AI দ্বারা সুনির্দিষ্ট পুনরুদ্ধার এবং ব্যবহারের জন্য। ফোল্ডার এবং নলেজ বেস অনুযায়ী নথি সংগঠন সমর্থন করে।

### Skill Manager — AI Responds at Command

## সার্ভার মোড ডিপ্লয়মেন্ট

ChatClaw সার্ভার মোডে চালানো যায় (ডেস্কটপ GUI প্রয়োজন নেই), ব্রাউজারের মাধ্যমে অ্যাক্সেসযোগ্য।

### বাইনারি সরাসরি রান

আপনার প্ল্যাটফর্মের জন্য বাইনারি [GitHub Releases](https://github.com/chatwiki/chatclaw/releases) থেকে ডাউনলোড করুন:

|| প্ল্যাটফর্ম | ফাইল |
||----------|------|
|| Linux x86_64 | `ChatClaw-server-linux-amd64` |
|| Linux ARM64 | `ChatClaw-server-linux-arm64` |

```bash
chmod +x ChatClaw-server-linux-amd64
./ChatClaw-server-linux-amd64
```

আপনার ব্রাউজারে http://localhost:8080 খুলুন।

সার্ভার ডিফল্টভাবে `0.0.0.0:8080` এ শোনে। আপনি এনভায়রনমেন্ট ভেরিয়েবলের মাধ্যমে হোস্ট এবং পোর্ট কাস্টমাইজ করতে পারেন:

```bash
WAILS_SERVER_HOST=127.0.0.1 WAILS_SERVER_PORT=3000 ./ChatClaw-server-linux-amd64
```

### Docker

```bash
docker run -d \
  --name chatclaw-server \
  -p 8080:8080 \
  -v chatclaw-data:/root/.config/chatclaw \
  registry.cn-hangzhou.aliyuncs.com/chatwiki/chatclaw:latest
```

আপনার ব্রাউজারে http://localhost:8080 খুলুন।

### Docker Compose

একটি `docker-compose.yml` ফাইল তৈরি করুন:

```yaml
services:
  chatclaw:
    image: registry.cn-hangzhou.aliyuncs.com/chatwiki/chatclaw:latest
    container_name: chatclaw-server
    volumes:
      - chatclaw-data:/root/.config/chatclaw
    ports:
      - "8080:8080"
    restart: unless-stopped

volumes:
  chatclaw-data:
```

তারপর রান করুন:

```bash
docker compose up -d
```

আপনার ব্রাউজারে http://localhost:8080 খুলুন। থামাতে: `docker compose down`। ডেটা `chatclaw-data` ভলিউমে স্থায়ী হয়।
