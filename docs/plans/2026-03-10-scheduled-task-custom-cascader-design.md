# 计划任务自定义时间级联选择设计

## 目标

在计划任务新建弹窗的“自定义时间”页卡中，调整频率选择区域的交互与文案，使其更接近级联选择体验。

## 需求确认

- 去掉“执行频率”文案。
- 选择“每天执行”时，移除右侧“每天执行”标题。
- 选择“每天执行”时，移除“每天会在所选时间执行一次”文案。
- 选择“每周执行”时，右侧使用类似级联选择的小弹层，单选周几。
- 选择“每月执行”时，右侧使用类似级联选择的小弹层，单选几号。

## 方案

- 保留左侧 `每天执行 / 每周执行 / 每月执行` 三个模式按钮。
- 右侧顶部继续保留时间输入。
- `daily` 模式下，右侧仅显示时间输入，不再显示任何说明块。
- `weekly` 模式下，右侧增加一个单选触发按钮，默认展示当前选中的 `周一/周二...`，点击后展开小面板供单选。
- `monthly` 模式下，右侧增加一个单选触发按钮，默认展示当前选中的 `1号/2号...`，点击后展开小面板供单选。
- 小面板为弹层式列表，不修改现有表单数据结构。

## 数据与约束

- `weekly` 继续使用 `form.customWeekdays[0]` 保存单选结果。
- `monthly` 继续使用 `form.customDayOfMonth` 保存单选结果。
- 不新增后端字段，不修改 `buildPayload` 映射。

## 验证

- 弹窗可正常打开关闭。
- 自定义时间下三种模式可正常切换。
- `weekly` 和 `monthly` 的弹层可展开、单选、回填并收起。
- `vue-tsc` 与 `vite build` 通过。
